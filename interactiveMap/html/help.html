Help: InteractiveMap

Requirements:A browser that supports HTML5 and the Canvas (Check it out!).
How to:To use the InteractiveMap-Interface you first have to activate it via the "activate IA"-Button above the map.
Then simply select a unit by clicking on it's occupied territory. Next select the order you want to give with one of the buttons next to the "activate IA"-/"deactivate IA"-Button or alternatively with the buttons that appeare next to your cursor on the map. Depending on the order type you selected you probably have to select one or more other territories to complete your order. You will find detailed information below this text.
If you want to start a new order, but have not finished the order before, click on the "Reset-Order"-Button before you start the new order. In the "Order-Line" there should appear "-NEXT ORDER-".
You can deactivate the InteractiveMap-Interface at any time.
After you have set your orders, you have to save them as normal. ALWAYS check if there are incomplete orders which won't be saved!
You can enter orders via the drop-down lists at any time. To let them appear on the map, make sure the InteractiveMap-Interface is activated and click on "Reset-Order"-Button. The "Reset-Order"-Button will always redraw the order-arrows with the information set in the drop-down lists.
Some additional features like greyed-out territories or a large map without scrollbars can be set in the options. The options can be entered by pressing the "Opt."-Button beneath the "?"-HELP-Button.
HOLD:Select the unit that should hold and press the "HOLD"-Button.
MOVE:Select the unit that should move, press the "MOVE"-Button and select the territory the unit should move to.
If you have selected a fleet that should move to a territory with two or more coasts that can be reached by the fleet, try to click on the coast the fleet should move to. Perhaps you have to try different points until you get the move you wanted. You could also set it via the drop-down list. Note that the selected coast is detected by the distance to the coast-coordinates (the positions of the map where fleets at coasts are placed).
If an army could reach the target territory "via convoy" or "via land", "via land" is set.
The normal movement-rules still apply!
SUPPORT HOLD:Select the unit that should support another unit holding, press the "SUPPORT HOLD"-Button and select the supported unit.
SUPPORT MOVE:Select the unit that should support another unit moving, press the "SUPPORT MOVE"-Button, select the target territory and THEN the origin territory.
IMPORTANT: Not that the order for the supported unit will be changed as well, if it was different before and if it is one of your own units. This is unlike the normal way to set the orders!
If the supported unit could move "via convoy" or "via land", "via land" is set. If "via convoy" is set before, it will stay "via convoy" as the MOVE-Order has not been different before.
Note that the same rules for fleets moving to coasts apply as with the MOVE-Order.
CONVOY:The way to set the CONVOY-order via InteractiveMap-Interface differs most from the normal way:
First select the army that should be convoyed, press the "CONVOY"-Button, then select the convoying fleets in the correct order and finally select the target territory of the army. The orders for the army and every fleet involved in the convoy will be set automatically, if they are your units.
This is the only time, you can select a foreign unit before you give an order via a button (necessary if you want to order your fleets to convoy a foreign unit from X to Y).
REATREAT / DISBAND:Units that have to retreat are drawn by a red "F" or "A" on a white rectangle. Perhaps you have to deactivate the interface to see the occupying unit as the letters are drawn on it.
The "DISBAND"- and "RETREAT"-Buttons act similar to the "HOLD"- and "MOVE"-Buttons. A DISBAND-Order appears as a small explosion next to the unit-sign on the map.
The normal movement-rules for the retreat phase still apply.
BUILDSelect an own, unoccupied Supply Center and press the "BUILD FLEET"- or "BUILD ARMY"-Button. A red "F" or "A" should appear next to a star on the selected territory.
To reset a BUILD-Order for a Supply Center, simply select the Supply center and press the "WAIT"-BUTTON. If you try to set more builds than you are allowed, the order at the top of the drop-down lists will be overwritten with the new order.
DESTROYSelect an own unit and press the "DESTROY"-Button. A big explosion should appear on the the unit.
To reset a DESTROY-Order, simply select a different unit. The "oldest" order will be overwritten if there is no free order left.


OPTIONS
"Toggle scrollbars on map"Adds or removes scrollbars, so large maps can be displayed on the whole screen or only in a small frame.
"Activate territory-grey-out"If activated, territories which can not be accessed for a selected unit with a selected order will be greyed-out.


TROUBLESHOOTING
Interface not visible (no buttons etc)The interface is probably not activated for this variant. Please contact an admin of this webdip-site.
During the loading of the page another window opens, which display some error messages.The map of the variant can not be modified for the interface. Please report the error messages to an admin.
After activation of the interface, you are not able to select any territories.Perhaps you clicked on a point on the map, which is not assigned to a territory. Try again, please!
If the problem still exists, please report this bug here or in the forum of this site.
Something elseAs the project is in alpha-phase, many issues could appear. Please report bugs here or in the forum of this site!
